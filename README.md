# Ian Eisele - Portfolio

Personal portfolio website built with Astro and Tailwind CSS v4.

**Live site:** [ianeisele.com](https://ianeisele.com)

## Development

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Updating Content

Edit the JSON files in `src/data/` to update portfolio content:
- `experience.json` - Work history
- `skills.json` - Skill categories
- `projects.json` - Project cards

## Tech Stack

- [Astro](https://astro.build) - Static site generator
- [Tailwind CSS v4](https://tailwindcss.com) - Utility-first CSS
- [Lucide Icons](https://lucide.dev) - Icon library
